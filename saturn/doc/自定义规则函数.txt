1.映射机制
	用规则自定义函数来实现。
	规则自定义函数可以为一段规则块指定一个名字，后期使用该规则块，引用该名字即可。
	规则自定义函数可以指定参数。
2.规则自定义函数定义语法
	func name(para...)
	{}//函数描述
	{//函数体
	 }
	
3.调用规则自定义函数
 	1.通过函数名调用:
	name()
	2.通过映射调用 
	<description{para}>
	
4.举例
	为如下规则定义一个函数：
	POS消费每300元积1分。
	
	函数定义：
	function pospoint(amount,point)
	{POS消费每{amount}元积{point}分}
	{
	
	}
	
	函数调用：
	1.通过函数名调用:
	pospoint(300,1)
	2.通过映射调用 
	<POS消费每{300}元积{1}分>
