


网银交易客户数： count(客户), 交易.金额 > 0
网银交易笔数：count(交易), 交易.金额 > 0
网银最高交易额：max(客户.客户交易额)
客户网银交易额: sum(交易.金额),交易类型 = '网银',客户

1.类别
  * 原子指标
  * 组合子标 通过对原子指标进行一定的计算得来。

2.定义
metric name
{frequency : //指标计算频率
   des: //描述，对该指标的描述. optional
}
{
	//实现体 指标的计算公式
}

3.实现
	指标的实现分为：
	1.指标定义的解析
	2.指标定义的存储和查找
	3.指标值的计算
	4.指标值的存取
	其中规则引擎负责指标定义的解析和指标值计算，由指标管理系统负责指标的其它方面。

4.指标值的存储
	所有指标值都存储在一个指标表中，每行记录表示一个指标值。
	*字段
	指标名 指标值 计算时间
