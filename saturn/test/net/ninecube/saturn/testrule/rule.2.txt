/* 持卡人在生日的当月刷卡消费可以获得双倍积分  */  
if 客户.出生日期.月份 == POS消费.交易日期.月份
	积分.消费积分 *= 2

/**
 * 凭信用卡消费或取现1元人民币（或1美元），即可获得1分（或8分）的消费积分，
 *  在汽车类商户每消费100元人民币积8分（每消费100美元积64分），
 * 在房地产类商户每消费100元人民币积6分（每消费100美元积48分）。 
 */
switch {
	case POS消费.商户类型 == "房地产类" :
		按比例积分(积分.消费积分, POS消费.消费金额, 100, 6)
	case POS消费.商户类型 == "汽车类" :
		按比例积分(积分.消费积分, POS消费.消费金额, 100, 8)
	default :
		按比例积分(积分.消费积分, POS消费.消费金额, 1, 1)
}
